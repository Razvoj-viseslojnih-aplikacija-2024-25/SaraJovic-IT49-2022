O aplikaciji

Sudski informacioni sistem je višeslojna web aplikacija razvijena u cilju digitalizacije i modernizacije upravljanja sudskim procesima. Sistem omogućava efikasno vođenje evidencije sudova, predmeta, učesnika i ročišta na jednom mestu.

Aplikacija je izgrađena korišćenjem Spring Boot tehnologije na serverskoj strani, koja obezbeđuje robustni REST API za upravljanje podacima, dok Angular framework na klijentskoj strani pruža dinamičan i intuitivan korisnički interfejs. Podaci se čuvaju u PostgreSQL relacionoj bazi podataka.
