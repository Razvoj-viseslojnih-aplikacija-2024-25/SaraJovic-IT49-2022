Autor aplikacije

Moje ime je Sara Jović, studentkinja 4. godine Inženjerstva informacionih sistema na Fakultetu tehničkih nauka. Ovu aplikaciju sam razvila u okviru predmeta Razvoj višeslojnih aplikacija. Projekat predstavlja implementaciju višeslojne arhitekture korišćenjem savremenih tehnologija za razvoj web aplikacija.

Sistem je realizovan kao studentski projekat sa ciljem primene stečenih znanja iz oblasti razvoja backend servisa pomoću Spring Boot framework-a i izrade korisničkog interfejsa korišćenjem Angular framework-a.
